Landing Page – Design e Desenvolvimento Web

Desenvolvida com W3.CSS para responsividade e rapidez, esta landing page foi construída para uma atividade da disciplina de DS1 do curso técnico de Desenvolvimento de Sistemas - ETEC. O layout dark e tecnológico valoriza títulos, chamadas à ação e seções bem organizadas.

PROCESSO DE DESENVOLVIMENTO

O desenvolvimento dessa LP — Landing Page — foi uma experiência completa: da idealização à publicação no GitHub Pages. Toda a base teórica que venho construindo foi colocada à prova ao estruturar semanticamente o HTML (indexação), aplicar classes do framework W3.CSS para estilizar a página e acrescentar interatividade com a lógica do JavaScript.

RECURSOS USADOS

TECNOLOGIAS

  • JavaScript (interações | smooth | slide).
  • HTML5 (semântica | indexação | atributos).
  • W3.CSS (grid | responsividade | utilitários).
  • CSS (flexbox | hover | status | transições | propriedades).

FERRAMENTAS

  • VSCode (codificação).
  • Git/GitHub (repositório local/público | hospedagem - pages).
  • Chrome DevTools (console | elementos).
